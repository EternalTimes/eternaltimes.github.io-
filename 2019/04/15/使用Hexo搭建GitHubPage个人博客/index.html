使用Hexo搭建GitHub Page个人博客

1.Hexo简介

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

2.安装及配置环境

Hexo需要Node.js（需要Node.js 6.9以上）支持
要使用Github Page来托管博客网页则需要Git支持

确认安装完这两个程序后，就可以开始安装Hexo了

2.1 安装Node.js

打开Node.js官网下载，下载后安装。
一直点击“Next”，不赘述，但
注意：记得勾选Path，如图

然后还是”Next”

2.2 安装Git

和安装Node.js一样，一直“Next”

2.3 安装Hexo

在桌面或者其他文件夹右键，选择Git Bash Here
在Bash中输入
cd d:/hexo 定位到你想搭建博客的路径
npm install -g hexo-cli 安装hexo
出现+ hexo-cli@1.1.0的提示就说明安装完毕
hexo init blog 初始化到当前目录的名为“blog”的子目录，安装所需包
如图出现”Start blogging with Hexo”即初始化完毕

P.S. 本人喜欢用Powershell，而不是Git Bash，所以界面有所不同，但操作是一样的。
